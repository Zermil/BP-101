# BP-101

[Lissajous Curves](https://en.wikipedia.org/wiki/Lissajous_curve).

Made with: 
- [SDL2](https://www.libsdl.org/)
- [ImGUI](https://github.com/ocornut/imgui)

## Build and Run

Build (remember to change "MSVC_PATH" in the build script)

```console
> build.bat
```

Run

```console
> cd build
> curves.exe
```
